## 3. 主要用語（まず押さえる10語）

| 用語 | 意味 | 補足 |
|------|------|------|
| **ゼネコン** | 総合建設会社。元請。鹿島・大林・大成・清水がスーパーゼネコン4社 | 躯体＋設備を一括発注。設備は下請に発注 |
| **完成工事高** | 建設会社の「売上」に相当。完成した工事の請負金額 | **設備会社**は完成工事高。ゼネコンは売上高。混同注意 |
| **躯体（くたい）** | 建築の骨組み（鉄骨・鉄筋コンクリート等） | 設備（電気・空調等）と対比 |
| **設備** | 電気・空調・衛生・昇降機等。DC・工場では建設費の約75% | 電気が設備の約50% |
| **経審（経営事項審査）** | 公共工事の入札参加資格。国交省が審査 | 経審が通らないと公共入札に参加できない |
| **選別受注** | 採算の合う案件だけ受ける | ゼネコン4社の戦略 |
| **電力系** | 電力会社の関連会社。関電工・九電工・トーエネック・ユアテック | **きんでんは電力系ではない**（独立系） |
| **DC** | データセンター。AI・クラウド用の施設 | 建設費の75%が設備。大型は1棟数百億円 |
| **成長セグメント** | 物流・工場・DC・再生エネルギー | 補助金・国内回帰・脱炭素で需要拡大 |
| **監理技術者・主任技術者** | 施工管理の資格者。現場の技術責任者 | 経審・入札の配置要件。人手不足で希少 |

※詳細は [18-company-meeting-essentials.md](./18-company-meeting-essentials.md) セクション1。

## 4. 成長セグメントの具体像（抽象→具体）

| セグメント | 具体例 | 規模感・補足 |
|------------|--------|--------------|
| **物流** | EC倉庫、3PL倉庫、冷凍冷蔵倉庫 | 首都圏・関西に集中。EC・宅配需要で拡大 |
| **工場** | 半導体工場、EV・蓄電池工場、食品・医薬品工場 | 補助金（半導体基金等）で大型案件。九州・東北で立地増 |
| **DC（データセンター）** | クラウド・AI用のサーバー施設。NTT大阪第7DC（470億円）、北九州DC（1,250億円）等 | 建設費の75%が設備。電気50%、空調25% |
| **再生エネルギー** | 太陽光、風力、蓄電池。自家消費型太陽光＋蓄電池＋PPAが主流 | FIT→FIP移行で売電価格急落。蓄電池はDC・工場・系統用でも拡大 |

## 5. バリューチェーン（誰が誰に発注するか）

```
発注者（国・自治体・民間企業）
    ↓
ゼネコン（元請。鹿島・大林・大成・清水等）
    ↓
一次下請（躯体・設備の専門会社）
    ├─ 躯体：鉄骨、鉄筋コンクリート等
    └─ 設備：電気（きんでん等）、空調（高砂熱学等）、管工事
         ↓
    二次下請（さらに専門化した会社）
```

※設備会社はゼネコンの**下請**。ただし大手設備会社（きんでん7,051億円等）は規模が大きい。

## 6. 建設業の制度（許可・経審の基本）

| 制度 | 意味 | 具体例・補足 |
|------|------|--------------|
| **建設業許可** | 建設工事を請け負うための許可。国交省・都道府県が交付 | 許可がないと建設工事を請け負えない。特定（元請）と一般がある |
| **経審（経営事項審査）** | 公共工事の入札参加資格。国交省が審査 | 経審が通らないと公共入札に参加できない。技術者（監理技術者・主任技術者）の配置が要件 |
| **総合評価入札** | 価格だけでなく技術・実績・地域貢献も評価する入札方式 | 「価格だけ」と軽視しない。技術力・実績が評価される |
| **監理技術者・主任技術者** | 現場の技術責任者。施工管理技士等が該当 | 経審・入札の配置要件。1級で監理技術者、2級で主任技術者。人手不足で希少 |
