# きんでん（1944・東証プライム）企業詳細

総合設備エンジニアリング最大手。**電力系ではない**独立系。IR・株主向け説明資料に基づく分析。

---

## 1. 企業概要

| 項目 | 内容 |
|------|------|
| 正式名称 | 株式会社きんでん |
| 設立 | 1944年 |
| 本社 | 大阪市北区 |
| 市場 | 東証プライム |
| 事業 | 総合設備エンジニアリング。配電・送電・発変電、建築電気設備、環境関連、情報通信、内装 |
| 完成工事高（2025年3月期） | 7,051億円（業界最大） |
| 営業利益率 | 8.6% |

**業界での位置づけ**: 電気設備5社の一角であり、唯一の**電力系ではない**独立系。業界最大規模。

---

## 2. 経営方針・ビジョン

### 2.1 中期経営計画「Sustainable Growth 2026」

- **テーマ**: ～人、心、そして未来へ～
- **計画期間**: 2021～2026年度
- **目指す姿**: 人財を軸とした成長投資による中長期的な利益創出能力の強化。建設業界・市場の環境変化においても**持続的成長・発展**。社会インフラを支える企業としての責任。

### 2.2 経営の基本方針

| 方針 | 内容 |
|------|------|
| **人を中心とした経営** | 「人と心」を経営の根幹に置き、人財を軸とした事業基盤の整備・強化 |
| **持続的成長** | 長期的視点に立ち、社会インフラを支える企業として持続的成長を目指す |
| **資本効率** | ROE7%以上、株主還元（配当性向40%目安）、政策保有株式縮減 |
| **ESG・SDGs** | 環境に優しく、持続可能でより良い社会の実現に貢献 |

## 3. 戦略の前提

### 3.1 市場環境認識

- 建設投資拡大、民間非住宅（DC・工場・半導体）旺盛
- 電力インフラ更新、省エネ・ZEB・脱炭素需要
- 再生可能エネルギー工事、洋上風力発電設備の成長

### 3.2 成長ドライバー

- 半導体・データセンター・蓄電池・再エネ
- 都市再開発、老朽設備更新
- 超大型工事、万博・IR関連工事

### 3.3 構造課題への対応

- 建設業就業者減少、人手不足
- 資材・人件費高騰、2024年問題
- → 生産性向上、施工技術革新、人財強化
