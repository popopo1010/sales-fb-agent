# ユアテック（1934・東証プライム）企業詳細

東北電力系電気設備大手。IR・株主向け説明資料に基づく分析。

---

## 1. 企業概要

| 項目 | 内容 |
|------|------|
| 正式名称 | 株式会社ユアテック |
| 設立 | 1934年（1944年設立説もあり） |
| 本社 | 宮城県仙台市 |
| 市場 | 東証プライム |
| 系列 | **東北電力系** |
| 事業 | 電気設備、空調設備工事 |
| 売上高（2025年3月期） | 約2,572億円 |
| 従業員数（連結） | 5,841名 |

**業界での位置づけ**: 電気設備5社の一角。東北電力の関連会社。DC・民間建設好調。

---

## 2. 経営方針・ビジョン

### 2.1 中期経営計画（2024-2028）

- **数値目標見直し**: 2025年10月に目標を更新
- **売上高目標**: 3,000億円（2030年代前半）
- **営業利益目標**: 200億円（2028年度目標を前倒し）
- **ROE**: 8.0%（2024年度実績で達成済み、高水準維持を目指す）

### 2.2 経営方針

- 東北電力グループとしての安定基盤
- 4事業領域での成長
- 統合報告書で財務戦略・人財戦略・サステイナビリティを公表

---

## 3. 戦略の前提

### 3.1 事業環境認識

- 民間建設投資の堅調
- 電力インフラ更新需要
- データセンター需要の好調
- 2025年3月期は電気設備5社として増収増益・過去最高

### 3.2 4つの主要事業

| 事業 | 内容 |
|------|------|
| 営業本部 | オフィスビル、工場、病院等の電気・空調・情報通信設備 |
| 電力インフラ本部 | 送電・配電・発変電設備、土木建築工事 |
| 再生可能エネルギー事業本部 | 太陽光・風力発電関連 |
| 海外事業 | ベトナム中心に約30年の施工実績。シグマ社を子会社化 |
