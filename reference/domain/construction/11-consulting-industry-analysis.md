## 2. 補完内容（要約）

| 項目 | 内容 |
|------|------|
| **バリューチェーン** | 発注者→元請（3〜5%）→一次下請・設備（5〜8%）→二次下請（2〜5%）。DC・工場は設備の取り分大 |
| **競争** | 大手50社で国内受注約24%。電気設備5社で相当シェア。→ company-ir/02 |
| **地域** | 首都圏：物流・DC。九州・東北：工場・再エネ。地方：公共中心 |
| **発注者** | 民間デベロッパー、企業、自治体、電力・通信（系列内） |
| **人材** | 就業者約500万人、55歳以上36.7%、2030年頃80万人減少試算 |
| **技術** | 2024年問題、2025年省エネ法改正、2026年FIT→FIP、2030年省エネ基準強化 |
| **リスク** | 補助金縮小、建設費高騰、技術者不足、供給過多、景気後退 |

## 3. 政策の方向性・タイムライン（いつ・何が・どう変わるか）

| 時期 | 政策・制度 | 建設業への影響 |
|------|------------|----------------|
| **2024年** | 時間外労働上限規制（2024年問題） | 残業抑制、生産性・省人化が必須 |
| **2025年** | 骨太方針2025、省エネ法改正検討 | 戦略的社会資本整備、GX投資、建築物の省エネ |
| **2026年** | FIT→FIP移行 | 再エネ建設の構造変化、自家消費型・蓄電池へ |
| **2030年** | 省エネ基準強化、GX官民150兆円 | ZEB・再エネ・蓄電池の建設需要。8大市場22.5兆円予測 |
| **継続** | 未来を拓くパートナーシップ | 元請・下請の取引適正化、付加価値向上 |
