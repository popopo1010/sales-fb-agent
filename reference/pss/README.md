# PSSマニュアル格納フォルダ

## 格納場所

**このフォルダ（`reference/pss/`）に PSS 営業スキームのマニュアルを格納してください。**

## 推奨ファイル構成

| ファイル | 説明 |
|----------|------|
| `pss-sales-scheme.md` | PSS営業スキームのメインドキュメント |
| `evaluation-criteria.md` | 評価基準・チェック項目（任意） |
| `flow.md` | 面談フロー・ステップ（任意） |

## 形式

- Markdown（.md）を推奨
- AIが読みやすいよう、構造化（見出し・箇条書き）を推奨

## 関連

- 詳細は [使用定義書](../../docs/01-usage-definition.md) を参照
- オペレーションマニュアルは `reference/operations/` に格納
