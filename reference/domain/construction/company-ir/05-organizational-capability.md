# Organizational Capability（組織能力）

マッキンゼー流：**7Sフレームワーク**で組織の実行基盤を評価。IR・業界動向から読み取れる組織課題を整理。

---

## 1. 7Sフレームワーク

| S | 要素 | 建設・設備業界の状況 |
|---|------|---------------------|
| **Strategy** | 戦略 | 成長セグメントへのシフト、採算改善。→ 04参照 |
| **Structure** | 組織構造 | 電力系は地域別、ゼネコンは事業部制。下請構造 |
| **Systems** | 制度・プロセス | 入札・経審、施工管理、品質管理。DX・BIM/CIM導入 |
| **Shared Values** | 共通価値観 | 安全、品質、納期。脱炭素・ESGの浸透 |
| **Skills** | スキル | 電気工事士・電験・施工管理技士。DC・半導体の専門技術 |
| **Staff** | 人材 | 就業者約500万人、55歳以上36.7%、29歳以下11.7%。2030年80万人減試算 |
| **Style** | リーダーシップ・文化 | 現場主義、技術重視。働き方改革・生産性向上 |

## 2. 組織課題（IR・白書から）

| 課題 | 内容 |
|------|------|
| **人手不足** | 技術者確保が経営課題。採用・定着・スキルアップが必須 |
| **2024年問題** | 時間外上限規制。残業抑制で生産性・省人化が必須 |
| **高齢化** | 55歳以上36.7%。知識承継、若年層採用が課題 |
| **生産性** | DX・省人化・施工効率化。BIM/CIM、ICT建機 |
| **協力会社** | 優良協力会社の確保。人手不足で取り合い |
