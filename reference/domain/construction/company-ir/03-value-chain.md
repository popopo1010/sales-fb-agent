# Value Chain（バリューチェーン）

マッキンゼー流：**ビジネスシステム**としてのバリューチェーン。利益構造と各プレイヤーの取り分を整理。

---

## 1. 建設業のバリューチェーン

```
発注者（民間デベロッパー・企業・自治体・電力・通信）
    │
    ▼
元請（ゼネコン・総合建設）
    │ マージン：3〜5%
    ▼
一次下請（躯体・設備）
    ├─ 躯体工事
    └─ 設備工事（電気・空調・衛生）… マージン：5〜8%
         │
         ▼
    二次下請 … マージン：2〜5%
```

---

## 2. 利益構造

| プレイヤー | 利益率目安 | 備考 |
|------------|------------|------|
| 元請（ゼネコン） | 3〜5% | 選別受注で採算重視 |
| 設備（電気・空調） | 5〜8% | DC・工場は設備の取り分大 |
| 二次下請 | 2〜5% | 人材・資材コストで圧迫 |

---

## 3. セグメント別の価値創出

| セグメント | 価値創出の焦点 | 設備の位置づけ |
|------------|----------------|----------------|
| **DC・工場** | 電力・冷却・信頼性 | 建設費の約75%が設備（電気50%、空調25%） |
| **再エネ** | 系統接続・採算・工期 | 送配電・発変電・蓄電池 |
| **オフィス・ビル** | ZEB・省エネ・品質 | 電気・空調・衛生 |
| **住宅** | スピード・コスト・ZEH | 電気・空調 |
| **公共** | 価格・総合評価 | 標準仕様、価格競争 |

---

## 4. 地域別・発注者別の違い

| 軸 | 特徴 |
|----|------|
| **地域** | 首都圏：民間非住宅40%、物流・DC集中。九州・東北：工場・再エネ。地方：公共中心 |
| **発注者** | 民間デベロッパー（納期・コスト）、企業（技術・補助金）、自治体（価格・総合評価）、電力・通信（系列内） |
