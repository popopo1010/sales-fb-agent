# 大気社（1979・東証プライム）企業詳細

空調・塗装・環境システムの総合企業。半導体・DC・自動車関連に強み。IR・株主向け説明資料に基づく分析。

---

## 1. 企業概要

| 項目 | 内容 |
|------|------|
| 正式名称 | 株式会社大気社 |
| 設立 | 1979年 |
| 本社 | 東京都品川区 |
| 市場 | 東証プライム |
| 事業 | 空調・塗装・環境システム |
| 経常利益（2025年3月期） | 199億円（過去最高） |
| 強み | 半導体・DC・自動車関連 |

**業界での位置づけ**: 空調設備大手。**塗装事業**も保有（自動車メーカー向け）。環境システムで半導体・DCに強み。

---

## 2. 経営方針・ビジョン

### 2.1 10年プラン2035（2026年3月期～2035年3月期）

- **長期目標**: 2035年3月期
- **完成工事高**: 5,000億円
- **ROE**: 12%
- **DOE**: 5%

### 2.2 新中期経営計画（2026年3月期～2028年3月期）

10年プラン2035の枠組みの中で、3年間の中期計画を策定。

### 2.3 主要戦略の方針

| 方針 | 内容 |
|------|------|
| シナジー創出 | 環境システム事業と塗装システム事業の連携 |
| 重点注力市場 | 半導体・DC・自動車への集中投資 |
| グローバル成長 | 海外展開 |
| デジタル戦略 | 経営資源配分の最適化 |
| 人材戦略 | 制度・システム基盤の整備 |

---

## 3. 戦略の前提

### 3.1 市場環境認識（IRより）

| 地域 | 認識 |
|------|------|
| 海外 | 世界経済減速懸念あるも各メーカー設備投資は堅調 |
| 国内 | 半導体・自動車・DC投資が継続、都市圏再開発も堅調 |
| 環境システム | 半導体・DC需要高まり継続 |
| 塗装 | 自動車メーカーで設備更新・増産投資 |

### 3.2 成長ドライバー

- 半導体製造設備
- データセンター
- 自動車メーカー向け塗装
- 都市圏再開発

### 3.3 採算性改善

- 価格適正化・採算改善を進め経常利益過去最高達成
- 成長投資に伴う販管費増を利益でカバー

---

| DOE | - | 5% |

※2026年3月期は成長投資に伴う販管費増で経常利益は減益見込みも、受注は過去最高。

---

## 5. 事業ポートフォリオ

### 5.1 事業構成

| 事業 | 内容 | 強みセグメント |
|------|------|----------------|
| 空調 | 建築空調、産業空調 | DC、半導体 |
| 塗装 | 塗装システム | 自動車メーカー |
| 環境システム | クリーンルーム等 | 半導体、DC |

### 5.2 強み

- 半導体・DC・自動車の**3本柱**
- 塗装事業は他空調大手にない差別化
- 環境システムでの技術力

---

## 6. 財務・業績

- 2025年3月期: 経常利益199億円（過去最高）
- 採算性改善で利益率向上
- 2026年3月期: 受注過去最高、経常利益は販管費増で182億円見込み
