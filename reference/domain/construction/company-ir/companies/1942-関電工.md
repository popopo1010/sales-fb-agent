# 関電工（1942・東証プライム）企業詳細

関西電力系電気設備大手。IR・株主向け説明資料に基づく分析。

---

## 1. 企業概要

| 項目 | 内容 |
|------|------|
| 正式名称 | 株式会社関電工 |
| 設立 | 1942年 |
| 本社 | 大阪市北区 |
| 市場 | 東証プライム |
| 系列 | **関西電力系** |
| 事業 | 電気設備工事、空調衛生設備 |
| 完成工事高 | 約6,700億円 |
| 創立100周年 | 2044年 |

**業界での位置づけ**: 電気設備5社の一角。関西電力の関連会社。半導体・DCで成長。

---

## 2. 経営方針・ビジョン

### 2.1 目指す姿「グリーンイノベーション企業」

- **目標年**: 2044年創立100周年
- **スローガン**: 「さらにかわる。より豊かな未来をつくる」
- **中期計画**: 2024-2026年度 関電工グループ中期経営計画

### 2.2 経営の方向性

前中期（2021-2023）では、業務プロセス改革・施工技術革新による生産性向上、半導体・データセンターを中心とした成長分野への営業展開を実施し、**過去最高益を達成**。当初公表した目標を全て達成。

---

## 3. 戦略の前提

### 3.1 事業環境認識（IRより）

| 認識 | 内容 |
|------|------|
| **構造課題** | 建設業就業者数の減少が確実視される |
| **需要** | 老朽化が進む社会インフラ・建築設備の更新需要への対応 |
| **社会要請** | 脱炭素社会実現への貢献 |
| **必要対応** | 魅力ある会社づくり、従来以上の生産力強化、新たな脱炭素化ソリューションの開発・提供 |

### 3.2 5つの方向性と事業戦略

2024-2026年度中期経営計画では、**5つの方向性**のもと、事業戦略と経営基盤強化戦略を推進。

## 5. 事業ポートフォリオ・成長戦略

### 5.1 強みセグメント

- **半導体**: 成長分野。前中期で営業展開を強化
- **データセンター**: 成長分野。大型案件
- **電力インフラ**: 更新需要
- **脱炭素**: ZEB、省エネ、再エネ

### 5.2 経営基盤強化

- 魅力ある会社づくり
- 生産力強化
- 脱炭素化ソリューションの開発・提供

---

## 6. 財務・業績

### 6.1 直近動向

- 2025年3月期: 全社増収増益・過去最高
- 2026年3月期見通し: 売上7,350億円（前期比+9.4%）、営業利益800億円（同+37.2%）等の報道あり

---

## 7. リスク・課題認識

- 建設業就業者減少
- 資材・人件費高騰
- 脱炭素対応の技術開発
- 地域集中（関西圏）
