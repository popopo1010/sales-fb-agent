# ドメイン情報

営業FBエージェント・人材紹介で活用する業界ドメイン知識の蓄積庫。

---

## 目的

| 用途 | 活用方法 |
|------|----------|
| **営業面談** | 候補者への説明・提案の根拠データ |
| **CAオンボ・面談準備** | 19-onboarding-prex 等で業界知識を習得 |
| **FB生成** | OPS（PSS・オペレーションマニュアル）を参照。ドメインはプロンプト肥大化回避のため未参照 |
| **候補者アトラクト** | 魅力的な情報の整理・訴求設計 |

## 情報の扱い

- **出典**: 公開情報・IRを明記。面談時は「国土交通省のデータでは」等と出典を伝えると説得力が増す
- **更新**: 建設投資見通しは年度更新。受注統計は月次。年1回以上の見直しを推奨
- **補足**: 数値は概算・見込みを含む。詳細は各ファイル内のソース参照
