# セグメント別 情報収集テンプレート

法人ヒアリング等で、セグメントごとに収集すべき情報の要素。

---

## 1. 収集要素（要約）

| カテゴリ | 要素例 |
|----------|--------|
| **市場背景** | 政策・制度の転換期、市場ドライバー、ブルーオーシャン |
| **セグメント詳細** | 成長見通し、勝ち筋・ネック、今後の方向性 |
| **企業・事業** | 戦略、主要リスク、フェーズ別ロードマップ |
| **要ファクト確認** | 数値の裏付け、事例の確認 |
| **運用・採用** | 採用状況、業績・案件、懸念点・課題 |

---

## 2. 他セグメントへの適用

物流倉庫・工場・DC・住宅・オフィス・再エネ等にも上記要素を適用。セグメント概要は [03-segments.md](./03-segments.md) 参照。
