# 建設業界 市場構造

## 1. 市場の全体像

**建設投資 75.6兆円**（2025年度、前年+3.2%）。民間67%（50.4兆円）、公共33%（25.2兆円）。建築65%（49兆円）、土木35%（27兆円）。

- **民間**: 住宅16.4兆円、非住宅建築12.3兆円（★+12.9%）、補修15.7兆円、土木約8兆円
- **政府**: 建築約16兆円、土木約9兆円

※出典：国土交通省「令和7年度建設投資見通し」

### 1.1 バリューチェーン（発注の流れ・具体例）

| 階層 | 役割 | 具体例 |
|------|------|--------|
| **発注者** | 工事を発注する主体 | 国・自治体（公共）、民間企業（物流会社、半導体メーカー等） |
| **ゼネコン（元請）** | 工事を一括請け負い、下請に発注 | 鹿島・大林・大成・清水。躯体・設備を専門会社に発注 |
| **一次下請** | 躯体または設備の専門工事 | 躯体：鉄骨、鉄筋コンクリート。設備：電気（きんでん等）、空調（高砂熱学等） |
| **二次下請** | さらに専門化した工事 | 電気の一部、配管、内装等 |

※設備会社はゼネコンの**下請**だが、きんでん（完成工事高7,051億円）等は規模が大きい。

## 2. 建築・土木・電気設備の位置づけ

**電気設備は建築の一部ではなく、建築・土木の両方に横断する工種**。設備工事市場は約19兆円で、電気が約半数。成長セグメント（DC・工場・再エネ）ほど電気設備比率が高い。

| 主戦場 | 内容 | 電気系資格 |
|--------|------|------------|
| **建築物の設備** | ビル・工場・DC・病院等の屋内電気 | 電気工事士、電気施工管理、電験 |
| **土木・インフラ** | 送配電、発変電、再エネ、通信 | 電気工事士、電験、送配電技能 |

## 4. インフラ老朽化・国家戦略との接続

**インフラ老朽化**（骨太方針・国土強靱化）: 道路橋の2030年約55%、2040年約75%が建設後50年以上経過見込み。限られた人員・予算下での長寿命化・安全性確保が重点。→ 公共土木・補修需要の持続要因。

**政府の重点政策との接続**: 建設業は「国民を守る」（災害復興・賃上げ）、「地方を守る」（地方の建設投資・公共工事）に貢献。元請・下請の取引適正化は「未来を拓くパートナーシップ構築推進会議」の文脈。→ 08セクション5参照。

## 5. 都市再開発・リニューアルの背景

都市開発が進んでいる主な要因。オフィス・ホテルセグメントの需要をけん引（03参照）。

| 要因 | 内容 |
|------|------|
| **建物の老朽化** | 1980〜90年代（バブル期）建設のオフィスビルが築40〜50年で老朽化。中規模オフィスビル約4万棟が該当。設備・耐震・省エネ基準で建て替え・大規模改修需要 |
| **政府の都市再生政策** | 国交省が都市再生関連施策を推進。国際競争力強化、生活の質向上、ウォーカブルなまちづくり。市街地再開発事業・都市再生総合整備事業 |
| **インバウンド回復** | 訪日外国人増加でホテル建設・改修が活発。宿泊需要が都市部の開発案件を後押し |
| **ZEB・脱炭素** | GXの流れで省エネ・ZEB・脱炭素建築の需要増。業務用脱炭素改修・ZEB化の補助金（08参照）が建て替え・改修を後押し |
| **都市集中** | 市街地再開発の5割超が東京都に集中。都市圏は地価・需要が高く採算が取りやすい |

**課題**: 資材価格上昇・人手不足で、進行中の市街地再開発事業の約8割で工期延期または費用増が発生（日経等）。
