# Growth Strategy（成長戦略）

マッキンゼー流：**3 Horizons of Growth**（成長の3地平）で戦略を整理。各社の中期計画・戦略の前提を統合。

---

## 1. 3 Horizons of Growth

| Horizon | 内容 | 建設・設備業界の例 |
|---------|------|-------------------|
| **H1: Core** | 既存事業の強化・採算改善 | 選別受注、価格適正化、生産性向上 |
| **H2: Emerging** | 成長分野への展開 | 半導体・DC・再エネへの営業展開 |
| **H3: Options** | 将来の選択肢 | M&A、ストックビジネス、新事業 |

---

## 2. 戦略の前提（IRから集約）

### 2.1 市場環境

| 前提 | 内容 |
|------|------|
| 建設投資 | 75.6兆円、民間67%・公共33%。民間非住宅+12.9%でけん引 |
| 成長セグメント | 物流・工場・DC・再生エネルギーが旺盛 |
| 構造課題 | 人手不足（2030年80万人減試算）、2024年問題、コスト高騰 |

### 2.2 成長ドライバー

| ドライバー | 内容 |
|------------|------|
| 半導体・DC | 国内回帰・AI需要。大型案件・設備工事比率高 |
| 蓄電池・再エネ | FIT→FIP移行、自家消費型太陽光・PPA。補助金が後押し |
| 電力インフラ | 老朽化更新、送配電・発変電 |
| 都市再開発 | ZEB・省エネ改修 |
| 製造業 | EV・蓄電池工場、クリーンルーム需要 |

### 2.3 企業対応

| 対応 | 内容 |
|------|------|
| 採算改善 | 価格適正化、受注選別、利益率向上 |
| 生産性 | DX・省人化・施工効率化 |
| 成長分野 | 半導体・DC・再エネへの営業展開 |
| 投資 | M&A、ストックビジネス、設備投資 |
| 株主還元 | 配当性向40%目安、ROE目標 |
