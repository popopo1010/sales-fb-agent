## 3.5 再生エネルギー 自家消費型・PPAの背景

なぜ「自家消費型太陽光＋蓄電池＋PPA」が主流か。太陽光パネルだけでは補助金を十分に使えない理由、PPAの経済性を整理。

### 3.5.0 なぜFIT→FIP移行が起こったか（政策転換の背景）

| 要因 | 内容 |
|------|------|
| **再エネ賦課金の膨張** | FIT買取費用は賦課金で全需要家が負担。2023年度で買取費用約2.3兆円。家計・企業を圧迫し制度の持続可能性が課題に |
| **市場の歪み** | FIT認定量の約9割が事業用太陽光に集中。地熱・風力等に比べ太陽光が短期で投資回収でき、再エネのポートフォリオが偏った |
| **政策の転換** | 国による長期高額買取から、市場原理に即した自立型電源へ。FIPは卸市場価格＋プレミアムで、需給を意識した発電を促す |
| **自家消費への誘導** | 売電より自家消費の方が経済的。買電価格25〜30円/kWh vs 売電7〜9円/kWh。政策として「売らずに使って」系統負荷を減らす意図 |

### 3.5.1 なぜ自家消費型＋蓄電池＋PPAが主流か

| 要因 | 内容 |
|------|------|
| **売電価格の急落** | 卒FIT後は7〜9円/kWh、2026年新FIT制度では5年目以降8.3円/kWhに急落。買電価格は25〜30円/kWh以上。「売るより使う」方が同じ1kWhで3〜4倍お得 |
| **FIT→FIP移行** | 国は長期高額買取から脱却。再エネ賦課金抑制のため、売電依存モデルを政策的に縮小 |
| **蓄電池コスト低下** | リチウムイオン電池は10年で90%以上コストダウン。自家消費率向上の採算が成立 |
| **補助金の条件** | 自治体補助金（例：兵庫県）は「**FIT認定を受けないこと**」「**自家消費率30%以上**」が条件。太陽光パネル単体の売電型では対象外になる場合が多い |

### 3.5.2 太陽光パネルだけだと補助金が使いにくい理由

| 制度・補助金 | 太陽光単体（売電型） | 自家消費型 |
|--------------|----------------------|------------|
| **FIT（2026年新制度）** | 1〜4年目24円/kWh、5〜10年目8.3円に急落。売電モデルは5年目以降採算悪化 | 自家消費なら買電回避（25〜30円/kWh相当）で経済性維持 |
| **自治体補助金** | 「FIT不可」「自家消費率○%以上」が条件の自治体が増加 | 対象になりやすい |
| **民間再エネ・地域脱炭素** | 太陽光・蓄電池の組み合わせが対象 | 蓄電池併設で申請しやすい |

※太陽光単体でも使える補助金はあるが、**政策の方向性は自家消費型への誘導**。

### 3.5.3 PPAの経済性・回収

| 項目 | 内容 |
|------|------|
| **需要家の初期費用** | **原則ゼロ**。PPA事業者が設備を設置・所有・維持管理 |
| **需要家の支払い** | 発電した電気を買電より安い単価でPPA事業者から購入。毎月の電気料金は継続支払い |
| **「手出し実質0」の意味** | 初期費用は0。ただし**毎月の電気代は支払う**。買電より安くなるので「電気代削減」がメリット。「回収」は需要家の概念ではなく、PPA事業者が設備投資を回収 |
| **契約期間** | 15〜20年。契約終了後、設備が無償譲渡される場合が多い |
| **補助金** | PPA事業者が申請。受け取った補助金は顧客の月額料金に反映（割引）される |

※需要家にとってのメリットは「初期投資なしで、買電より安い電気を使える」こと。工場・倉庫などでは補助金活用で5〜7年で投資回収する自己所有モデルもあり、資金力に応じて選択。

## 3.6 データセンター 規模感・代表例（実際の金額）

DCの規模感がわかるよう、代表的な事例を一覧化。建設費の約75%が設備（電気約50%、空調・衛生約25%）。

### 3.6.1 代表的なDCの規模・建設費一覧

| 事例 | 投資額（建設費） | 規模 | 備考 |
|------|------------------|------|------|
| **NTT Com 大阪第7DC** | **470億円** | IT電力35.5MW、SVルーム11,500m²、5,000ラック | 複数棟。1期1〜2棟目2019年竣工、6棟目まで計画 |
| **APL 北九州DC** | **1,250億円** | 受電容量120MW（九州最大級）、敷地62,822m² | 2027年秋着工予定 |
| **電算システム 東濃IDC増築** | **約3.5億円** | 28ラック、建築面積110m² | モジュール型。中小規模の代表例 |
| **IDCフロンティア 白河5号棟** | 数十億円規模 | 約3,200m²、約1,400ラック | 中規模棟の代表 |
| **さくら 石狩DC** | 数百億円規模 | 最大8棟・4,000ラック | 郊外型。PUE 1.11 |

### 3.6.2 建設費の内訳（空冷式DCの目安）

| 項目 | 割合 | 内容 |
|------|------|------|
| **電気設備** | 約54% | 受変電、UPS、配電、サーバーラック用電源 |
| **機械設備（空調等）** | 約22% | 空調・冷却、衛生設備 |
| **建築（コア/シェル）** | 約14% | 躯体、外装 |
| **その他・予備費** | 約10% |  |

※設備合計で約75%。オフィス（約30%）より設備比率が圧倒的に高い。2021〜24年で建設費約45%増、設備工事費上昇が直撃。

### 3.6.3 規模別の目安

| 規模 | 投資額目安 | ラック数 | 用途 |
|------|------------|----------|------|
| **超大型** | 500億〜1,000億円超 | 数千〜1万ラック級 | クラウド事業者、ハイパースケーラー |
| **大型** | 100億〜500億円 | 1,000〜5,000ラック | 大手DC事業者 |
| **中規模** | 10億〜100億円 | 数百〜1,000ラック | 地域DC、企業向け |
| **小規模** | 数億円 | 数十〜数百ラック | エッジDC、増築 |

## 3.7 蓄電池 用途別セグメント・拡大領域

蓄電池は太陽光とセット以外にも複数の用途で拡大。中カテゴリ別に整理。

### 3.7.1 蓄電池の3大用途カテゴリ

| カテゴリ | 規模目安 | 主な用途 | 拡大ドライバー |
|----------|----------|----------|----------------|
| **住宅用** | 4〜16kWh/戸 | 太陽光連携、停電時非常用、電気代削減 | FIT→FIP、ZEH+、補助金。太陽光とセットが主流だが単体も増加 |
| **産業用（C&I）** | 数十〜数百kWh、場合により数MWh | 工場・ビル・倉庫のピークカット、BCP、デマンドレスポンス | 電気料金高騰、BCP・ESG、補助金（民間再エネ、地域脱炭素） |
| **系統用** | 数MWh〜数十MWh、MWクラス | 再エネ変動吸収、周波数調整、需給調整、容量市場 | 再エネ増加、系統安定化、電力市場の自由化 |

### 3.7.2 太陽光以外で拡大している領域

| 領域 | 内容 | 建設・設備工事との接点 |
|------|------|------------------------|
| **データセンター** | UPS内蔵・バックアップ、ピークシフト。停電時1分間をカバーし非常用発電機起動まで | DC建設の電気設備。ゼロエミ化で再エネ＋蓄電池導入 |
| **工場・倉庫** | ピークカット、基本料金削減、BCP。大規模成長投資補助金の対象 | 工場建設・設備工事。九電工等が蓄電池開発案件へ出資し工事受注 |
| **系統用・送配電** | 電力会社・商社が設置。再エネ出力変動吸収、容量市場・需給調整市場で収益 | 送配電・発変電工事。きんでん・関電工等の電気設備5社が関与 |
| **V2H（車載電池）** | EVのバッテリーを住宅に供給。定置型とは別の「走る蓄電池」 | 住宅・EV充電設備。太陽光＋EV＋V2Hの組み合わせで拡大 |
| **マイクログリッド・地域** | 自治体・地域の自立電源。災害時の孤立運転 | 地域脱炭素交付金、民間再エネ。土木・電気設備 |

### 3.7.3 建設業界での蓄電池の位置づけ

| セグメント | 蓄電池の関わり |
|------------|----------------|
| **再生エネルギー** | 太陽光＋蓄電池が主流。自家消費型・PPAで拡大 |
| **工場** | 半導体・蓄電池**工場**の建設（経産省補助）＋工場**内**蓄電池（ピークカット・BCP）の2軸 |
| **データセンター** | ゼロエミ化・BCPで蓄電池導入。UPS・バックアップ |
| **住宅** | ZEH+で蓄電池が追加要件の1つ。定置型またはV2H |
| **送配電・系統** | 系統用蓄電池。再エネ増加に伴い需要拡大 |

※九電工は蓄電池・DC開発案件への**出資**で工事受注を確保する戦略。設備会社は「工事」だけでなく「案件開発・出資」にも関与。

## 4. 2030年展望・構造的課題

### 4.1 2030年市場展望

| シナリオ | 8大市場見通し | 主なトリガー |
|----------|---------------|--------------|
| **楽観** | 25兆円超 | 再エネ・DC・工場投資継続、補助金拡充 |
| **ベース** | 22.5兆円（現状24.6兆円比で約92%） | 人口減少・建設費高騰・規模縮小 |
| **悲観** | 20兆円以下 | 景気後退、事業延期・キャンセル、供給過多 |

※詳細シナリオは [12-segment-scenarios.md](./12-segment-scenarios.md) 参照。成長セグメント（物流・工場・DC・再エネ）へのシフトが企業戦略の焦点。

### 4.2 構造的課題（国土交通白書）

| 課題 | 内容 |
|------|------|
| **人手不足・就業者減少** | 2030年頃に建設業就業者が約80万人減少する試算 |
| **年齢構成** | 55歳以上36.7%、29歳以下11.7%。高齢化・若年層不足が進行 |
| **2024年問題** | 時間外労働の上限規制。残業抑制で生産性・採用が課題に |
| **労働環境** | 労働時間が長い、賃金が他産業より低め。採用競争力に影響 |
| **技術者確保** | 施工管理技士等の資格者不足。監理技術者・主任技術者の配置が経審・入札の前提 |

### 4.3 成長ドライバー vs ヘッドウィンド（02と整合）

| 成長要因 | 逆風要因 |
|----------|----------|
| 都市再開発・リニューアル、脱炭素・再エネ、インフラ老朽化、DC・物流需要、製造業国内回帰 | 資材・人件費高騰、人手不足・高齢化、建設費高騰、人口減少 |

### 4.4 人材紹介への示唆

- **優先訴求**: 楽観〜ベースのセグメント（物流・工場・DC・再エネ）を優先
- **課題の転換**: 人手不足＝スキルアップ機会、2024年問題＝働き方改革の追い風
- **候補者タイプ別**: 電気系→DC・工場・再エネ、施工管理→成長セグメントの大型案件。詳細は [04-recruitment-playbook.md](./04-recruitment-playbook.md) 参照
