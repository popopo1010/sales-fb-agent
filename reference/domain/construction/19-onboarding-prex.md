# 新規入社者向けオンボーディング資料（Prex）

電気関係・太陽光関係・施工管理（建設）のカバー領域における実務ノウハウ。業界リファレンス（00〜18）とあわせて活用。

※資格要件・必要年数は法改正で変動する場合あり。受験前年に最新の受験案内・講習要項を確認すること。

---

## 1. 事業前提・カバー領域

| 項目 | 内容 |
|------|------|
| **カバー領域** | 電気関係、太陽光関係、施工管理（建設）。施工管理はフェーズ1 |
| **取引法人** | 約1,500社。そのうちコア求人（初回で当てる求人）20社 |
| **コア求人エリア** | 東京、愛知、福岡、大阪に絞る |

## 2. 電気工事士「実務経験」として認められる基本条件

| 項目 | 要件・ポイント | 典拠 |
|------|---------------|------|
| **従事できる立場** | 第二種電気工事士免状もしくは認定電気工事従事者の範囲で合法的に作業していること（無資格作業はカウント不可） | 東京都電気工事工業組合 |
| **対象工事** | 電気工作物の**設置または変更工事**（配線・盤・機器取付け等）。設計・検査も含むが、製造や土木作業のみは除外 | 東京都電気工事工業組合 |
| **除外工事** | 軽微工事（電球交換など）、ネオン・非常用発電装置といった特殊電気工事、50kV以上の架空線路、保安通信設備、法令違反工事 | 東京都電気工事工業組合 |
| **経験期間の数え方** | 1日＝8h換算で**合算3年以上**（第一種電気工事士免状の場合）。パート勤務でも可。2022年改正で「5年→3年」へ短縮 | 工事士.com等 |
| **証明方法** | 勤務先（電気工事業登録業者など）の代表者が発行する「実務経験証明書」と工事実績一覧を免状申請時に提出 | - |

> **ポイント**: 経験に入るのは「資格を得てから」「合格通知を受けてから」の期間。合格前の無資格作業はカウントされない。

### 2.1 実務経験を積める代表的パターン

| 入口資格 | 積める現場例 | 必要年数 | 免状・認定申請先 |
|----------|--------------|----------|------------------|
| 第二種電気工事士 | 戸建・小規模店舗（50kW未満）の配線工事 | ―（合格即免状） | 都道府県知事 |
| 第二種+3年経験 **または講習修了** | ビル自家用（500kW未満）低圧部 | 3年 | 認定電気工事従事者を経産局へ申請 |
| 第二種+3年経験 **または電験等** | 500kW以上・高圧受電設備 | 3年 | 第一種電気工事士免状を都道府県知事へ申請 |
| 高圧電気工事技術者試験合格者 | 同上 | 3年 | 同上 |
| 電験三種免状所持者 | 工場・ビルの保安監督業務 | 5年 | 第一種電気工事士免状 |

### 2.2 実務経験にならない主な作業

- 電球交換・コンセントカバー交換など**軽微工事**
- ネオン工事／非常用発電装置（500kW未満）など**特殊電気工事**
- **50kV以上の架空電線路工事**
- 保安通信設備工事
- 無資格・無登録で行った違法工事一切

> **Tip**: 近年はEV充電器設置や再エネ事業で高圧・特高案件が増加。早めに第一種と1級施工管理技士を組み合わせると仕事の幅が広がる。

## 4. 実務経験証明の作成・申請の流れ

1. **証明者の確認**: 登録電気工事業者や電気主任技術者の下で従事した期間であることを会社印・証明者印で明記
2. **工事内容を具体的に**: 「〇〇ビル受変電設備更新（契約電力1,200kW）幹線張替え・動力盤取替」など、設置or変更であることが分かるよう記載
3. **期間集計**: 週4日勤務などの場合は日数合算表を添付し「合計○年○月○日」と記載
4. **提出先**: 都道府県窓口（第一種・第二種免状）または産業保安監督部（認定電気工事従事者・特種）

## 7. 面談フロー・トーク・提案方法

### 7.1 面談の前提

| 項目 | 内容 |
|------|------|
| **初回面談** | **企業を提案しないと失注する**（未契約求人でもあてる/duda） |
| **複数社提案** | 1社だけだと3ヶ月でやめてしまう。2〜3社を比較検討した方がよい |
| **時間** | 一人10〜20分くらい。施工管理は忙しい |
| **普段の業務時間** | いつだったらつながるかを聞いておくこと |

### 7.2 トーク内容（施工管理）

- どんな現場で、どんな規模で、どんな人数で施工管理をやっていたか
- その対応件数をどの程度でやっていたのか
- これまでの就業期間どれくらいあるか
- 同じ業界だったらトータルどれくらいいたか
- どこのプロジェクトにいたか

### 7.3 提案方法

| 項目 | 内容 |
|------|------|
| **コア求人** | 各都道府県でコア求人をあてることが重要 |
| **求人紹介** | 必ず電話面談実施。業界特徴・会社特徴・背景を踏まえ、求職者のキャリアにあっていることを口頭で説明 |
| **年収相場** | 相場を伝え、相場を超えている場合は「なぜ辞めたいのか？」を確認。レンジまで落とせるなら対応 |
| **単身赴任** | 年収高くないといけない。実家になるといける |

### 7.4 初回の流れ

1. 電話面談①：15分
2. 「合う求人紹介します」でLINE or メール
3. 「求人の特長説明させてください」で電話面談②を取る
4. 書類回収（上からやる）
5. 雛形を送って空欄うめてもらう / LINEで全項目送って回答 / 電話で全項目ヒアリングしてこちらで作る
6. ※回収できない方は別に原因ありと考える

## 9. 事業フェーズ・立ち上げのポイント

### 9.1 フェーズ別

| フェーズ | 内容 |
|----------|------|
| **Phase1** | 死ぬほど活動量担保して獲得しにいく。22:00まで架電し続ける |
| **Phase2** | 強い人が他社でも決定しなかった。名前があるところでやってもうまくいかなかった。900件CVを取っても各オーダー理解ができない。求人を聞いて当てるだけに。→法人を絞る |
| **Phase3** | 分母が減るが。CAひとりあたり新規60〜70件、そのうち初回面談35件（50%） |

### 9.2 事業立ち上げで重要なポイント ベスト3

1. **圧倒的行動量担保**（架電数100/日）
2. **メイン顧客の確保**: 徹底的に商談して1社理解を深める。数取れば取るほど沼
3. **法人区分け**: 経験ある/未資格、50〜60代遅れる、給与がいい、未資格/未経験

### 9.3 法人への条件緩和

- 資格持っていて経験ある人を求めている → 資格はあるけど未経験も取る
- 中堅取れていると若手も取り入れたほうがいい
- 未通電の対応
