## 4. 優先順位の考え方

1. **必須**: 入力→FB生成→Slack送信の一連フロー
2. **高**: 参照ドキュメント整備、プロンプト設計、Slack送信
3. **中**: 一括処理、リトライ、ログ
4. **低**: 履歴保存、高度な整形
