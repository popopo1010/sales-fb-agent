## 評価指示

上記の書き起こしを、PSSとオペレーションマニュアルに沿って評価し、以下の形式でフィードバックを出力すること。
書き起こし本文は出力に含めず、以下の8項目のみを出力する。

### 1. 良い点
- （具体的な箇所と理由）

### 2. 改善点
- （該当箇所の目安時間と具体的なアドバイス）
- ※除外ルールに該当する点は含めないこと

### 3. ニーズの整理
- **ニーズ**: 聞き取れたニーズの要約
- **具体的なニーズ**: ニーズの詳細・具体化
- **なぜそのニーズが重要なのか**: 候補者にとっての重要性
- **そのニーズが重要に至った背景**: 背景・経緯・理由

### 4. 意思決定の6カテゴリで重要になりそうな項目
オペレーションマニュアル第6章の「意思決定の6カテゴリ」を参照。①経済的 ②キャリア ③労働環境 ④組織・人間関係 ⑤心理的 ⑥社会的 のうち、この候補者にとって重要になりそうなカテゴリを2〜4つ、発言内容から根拠付きで記載。

### 5. 年収相場との整合性・許容アプローチ
下記の年収相場と候補者の資格・経験・希望年収を照合。相場との整合性を評価し、ずれがある場合は「どうすればその年収を許容させることができるのか」具体的なコミュニケーションのアドバイスを記載（OPS「条件が高すぎる時」のフレーズ、「希望は希望、現実の勝ち筋をあわせて設計」「選択肢を3つに分けて（挑戦/現実/安全）」等を参照）。ずれがない場合は「相場内で問題なし」と簡潔に記載。

| 資格・経験 | 年収レンジ（万円） |
| --- | --- |
| 資格なし | 360〜400 |
| 第2種電気工事士・経験なし | 360〜420 |
| 第2種電気工事士・現場経験あり | 400〜500 |
| 第1種電気工事士 | 450〜600 |
| 施工管理の経験あり | 600以上 |

### 6. 障壁になりそうなポイント
商談を進めていく上で、障壁になりそうなポイントがあれば記載する。
- 書き起こしに他社エージェントの介在が含まれる場合、OPS第6章「6-4bis. 他社エージェントが介在する場合」を参照し、他社エージェントを頼らない方針のアドバイスを含めること。

### 7. 総評・アドバイス
- （まとめ）

### 8. 残論点（裏のニーズ深掘りに特化）
- **深掘りすべきテーマ**: 裏のニーズ・背景が十分に聞けていない領域
- **具体的な質問フレーズ（言い回し）**: 候補者にそのまま使える質問を3〜5個、具体的に記載

## ルール

### 出力フォーマット
- 各項目の間には必ず1行空けて改行を入れる（視認性向上）
- 各項目の中では、箇条書きの要素ごとに改行を入れる（1行に詰め込まない）

### 改善点の除外
- 「2〜3分」と伝えて実際に30〜40分かかっても問題ない（OPS通り）。改善点として挙げないこと。

### 残論点の深掘り観点
- オペレーションマニュアルの「希望年収の理由を聞く時」「動機の深層を掘る」を参考
- 例：「仮に希望額の〇〇万円を実現できたら、今の生活の中で一番変えたいこと、叶えたいことは何ですか？」
- 例：「その条件を大事にしたい背景があるからですよね。もう少しだけ、そのあたりを教えていただけますか？」
