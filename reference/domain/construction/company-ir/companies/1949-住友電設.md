# 住友電設（1949・東証プライム）企業詳細

住友電工グループの電気設備。IR・株主向け説明資料に基づく分析。

---

## 1. 企業概要

| 項目 | 内容 |
|------|------|
| 正式名称 | 住友電設株式会社 |
| 設立 | 1947年（太陽電気工事）、1950年株式会社化、1985年現社名 |
| 市場 | 東証プライム |
| 系列 | **住友電工グループ** |
| 事業 | 電気設備、エネルギー、通信、プラント・空調 |
| 売上高（概算） | 約200億円 |

**業界での位置づけ**: 住友電工の施設工事を譲受。電力系ではない。送変電・通信・プラントに強み。

---

## 2. 経営方針・戦略の前提

### 2.1 事業沿革

- 設立時: 地中線・海底ケーブル工事
- 拡大: テレビ放送局アンテナ、送電線、海峡横断工事
- 現在: 電気設備、エネルギー、通信システム、プラント・空調

### 2.2 強み

- 住友電工グループとの連携
- 送変電・電力インフラ
- 通信システム
- プラント・空調

---

## 3. 戦略の前提

- 住友電工グループの投資に連動
- 電力インフラ更新
- 通信・データセンター需要
- 脱炭素・再エネ
