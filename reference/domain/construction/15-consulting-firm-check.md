# コンサル5社視点 チェック結果

建設業界ドメイン情報を、ボズ・アレン、カーニー、マッキンゼー、BCG、ベインの各社のフレームワーク・観点からチェック。

---

## 1. マッキンゼー（McKinsey）視点

### 1.1 充足している点 ✓

| フレーム | 所在 | 評価 |
|----------|------|------|
| **Issue Tree** | 01 | 市場規模・構造・成長・競争・見通し・示唆でMECEに分解 |
| **Pyramid Principle** | 01 | キーメッセージ1行→So What 4項目で結論ファースト |
| **5 Forces** | company-ir/01 | 業界構造分析。5力すべて評価済み |
| **PEST** | company-ir/01 | マクロ環境を4軸で整理 |
| **3 Horizons of Growth** | company-ir/04 | H1 Core / H2 Emerging / H3 Options で各社戦略整理 |
| **Value Chain** | company-ir/03 | 発注者→元請→下請の利益構造 |
| **7S** | company-ir/05 | Strategy, Structure, Systems, Shared Values, Skills, Staff, Style |
| **3C** | company-ir/02 | Customer, Competitor, Company |
| **Scenario Planning** | 12, company-ir/07 | 楽観・ベース・悲観の3シナリオ |

### 1.2 不足・要強化 △

| 論点 | 内容 |
|------|------|
| **So Whatの具体化** | 「何を・いつ・誰が」の優先アクションが抽象的。意思決定基準の明示が弱い |
| **定量の穴** | 8大市場のセグメント別金額が欠落（矢野有料のため）。e-Stat概算の追記余地 |
| **論点の深掘り** | 各論点の「So What」が1段階で止まっている。2〜3段の掘り下げが弱い |

## 2. BCG（Boston Consulting Group）視点

### 2.1 充足している点 ✓

| フレーム | 所在 | 評価 |
|----------|------|------|
| **成長×規模の2×2** | 03 | 高成長/低成長×規模のマトリクス。BCGマトリクスに近い |
| **セグメント別ポジション** | 03, 12 | 物流・工場・DC・再エネ＝成長、住宅・店舗＝成熟の整理 |
| **シナリオ** | 12 | 楽観・ベース・悲観で不確実性に対応 |
| **競争構造** | company-ir/02 | 寡占度・プレイヤー整理 |

### 2.2 不足・要強化 △

| 論点 | 内容 |
|------|------|
| **BCGマトリクス（成長率×相対シェア）** | 現状は成長×規模。各セグメントの「相対シェア」が未整理。Stars/Cash Cows/Question Marks/Dogs の分類なし |
| **Experience Curve** | 累積生産量とコストの関係。規模の経済・学習効果の定量が未記載 |
| **Strategic Imperatives** | 「やるべき3〜5つの戦略的必須事項」が明示されていない |
| **Time-based competition** | 納期・スピードを競争軸としての整理が弱い（04の訴求軸には「スピード」あり） |

## 3. ベイン（Bain & Company）視点

### 3.1 充足している点 ✓

| フレーム | 所在 | 評価 |
|----------|------|------|
| **顧客セグメント** | company-ir/02 | 発注者タイプ別（デベロッパー・企業・自治体・電力等）の整理 |
| **競争ポジション** | company-ir/02 | 電気設備5社の強み・弱み |
| **シナリオ** | 12, company-ir/07 | 各社の楽観・悲観 |
| **バリュープール** | company-ir/03 | 元請3〜5%、設備5〜8%等の利益構造 |

### 3.2 不足・要強化 △

| 論点 | 内容 |
|------|------|
| **Net Promoter Score / 顧客ロイヤルティ** | 発注者満足度・リピート率・紹介率の視点が未記載 |
| **Core Competence** | 各社・各セグメントの「中核的競争力」の明示が弱い。勝ち筋はあるが深掘り不足 |
| **Repeatable Model** | 成功パターンの「再現可能モデル」が未整理（例：DC案件の受注〜施工の成功公式） |
| **Results Delivery** | 実行・成果創出の観点。KPI・マイルストーンの追跡が未記載 |

## 4. カーニー（A.T. Kearney）視点

### 4.1 充足している点 ✓

| フレーム | 所在 | 評価 |
|----------|------|------|
| **サプライチェーン** | company-ir/03 | 発注者→元請→下請のバリューチェーン |
| **公共セクター** | 02, 08 | 公共33%、政府投資の内訳。補助金・政策 |

### 4.2 不足・要強化 △

| 論点 | 内容 |
|------|------|
| **Strategic Sourcing** | 発注者側の調達戦略（入札方式・総合評価・価格競争）の深掘りが弱い |

| **Assessment / 診断** | 業界・企業の「現状診断」フレームが未整備。スコアカード等 |
| **地域別オペレーション** | 首都圏・九州・東北の立地別の調達・施工の違いが要約レベル |

## 5. ボズ・アレン（Booz Allen Hamilton）視点

### 5.1 充足している点 ✓

| フレーム | 所在 | 評価 |
|----------|------|------|
| **Capability** | company-ir/05 | 7Sで組織能力を整理。Skills, Staff, Systems |
| **公共・政府** | 02, 08 | 公共33%、政府投資、補助金。Boozの政府系強みに合致 |
| **戦略アライメント** | company-ir/04 | 3 Horizonsで戦略の時間軸を整理 |
| **インフラ** | 02, 03 | 建設・土木・電気設備のインフラ位置づけ |

### 5.2 不足・要強化 △

| 論点 | 内容 |
|------|------|
| **Capability-driven Strategy** | 「どの能力を強化すべきか」の優先順位が未明示。能力ポートフォリオの整理が弱い |
| **Government / Public Sector 深掘り** | 公共調達の入札改革、総合評価、地域貢献等の詳細が薄い |
| **Technology & Innovation** | デジタル変革・技術ロードマップの詳細が11に要約のみ。i-Construction, BIM/CIMの深掘り余地 |

## 6. 共通の要強化項目（5社横断）

| 論点 | 推奨対応 |
|------|----------|
| **定量の穴** | 8大市場セグメント別金額の概算（e-Stat自集計 or レンジ）を03に追記 |
| **優先アクションの具体化** | 01のSo Whatに「3ヶ月でやるべきこと」「意思決定基準」を追加 |
| **M&A・事例** | ゼネコン・設備のM&A事例、参入・撤退事例を11 or company-irに追記 |
| **リスクマトリクス** | 発生確率×影響度の2軸でリスクを可視化（11に追加） |
| **競争軸の明確化** | 各セグメントの「勝ち筋」を価格・技術・関係性・スピードで整理 |
