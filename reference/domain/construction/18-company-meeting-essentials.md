# 企業代表との面談 必須理解事項

各企業の代表と話す際に、**理解していないと会話にならない**ポイントを整理。面談前の必須チェック。

※**上場企業の経営者**（社長・IR担当役員等）と話す場合は、セクション7「上場企業経営者との面談で必要な追加知識」も必須。

---

## 1. 業界用語（会話で必ず出る）

| 用語 | 意味 | 補足 |
|------|------|------|
| **ゼネコン** | 総合建設会社。元請。鹿島・大林・大成・清水がスーパーゼネコン4社 | 「ゼネコン」と言えば大手を指すことが多い |
| **一次下請・二次下請** | 元請の下で工事を受注する会社。一次＝躯体・設備、二次＝その下 | バリューチェーン。未来を拓くパートナーシップで取引適正化が政策 |
| **躯体（くたい）** | 建築の骨組み（鉄骨・鉄筋コンクリート等） | 設備と対比。躯体工事と設備工事は別 |
| **設備** | 電気・空調・衛生・昇降機等。建設費の一部。DC・工場では約75% | 電気が設備の約50% |
| **完成工事高** | 建設会社の「売上」に相当。完成した工事の請負金額 | **設備会社**は完成工事高が業界標準。ゼネコンは連結決算で売上高を使用 |
| **経審（経営事項審査）** | 公共工事の入札参加資格。国交省が審査 | 経審が通らないと公共入札に参加できない |
| **選別受注** | 採算の合う案件だけ受ける。利益率重視 | ゼネコン4社の戦略。単価の安い案件は受けない |
| **手持ち工事高** | 受注済みで未完成の工事の金額。今後の売上見込み | 「手持ちが堅調」＝今後の売上見込みが良い |
| **電力系** | 電力会社の関連会社。関電工（関西）、九電工（九州）、トーエネック（中部）、ユアテック（東北） | きんでんは電力系ではない（独立系） |
| **サブコン** | 設備工事の専門会社。空調・電気等 | ゼネコンの下請。高砂熱学、大気社等 |
| **ZEB・ZEH** | ZEB=ネットゼロエネビル、ZEH=ネットゼロエネ住宅 | 省エネ建築。補助金の対象 |
| **FIT・FIP** | FIT=固定価格買取、FIP=市場連動型。2026年FIT→FIP移行。売電価格急落で自家消費型・蓄電池・PPAが主流。蓄電池は太陽光以外にDC・工場・系統用で拡大（03セクション3.7参照） | 再エネの売電制度。建設需要に影響 |
| **監理技術者・主任技術者** | 施工管理の資格者。現場の技術責任者 | 施工管理技士等。1級累計：土木35〜40万、建築30〜35万、電気工事8〜10万、管工事5〜7万人（13参照）。人手不足で希少 |

## 2. 企業タイプ別「会話の前提」

### 2.1 ゼネコン（鹿島・大林・大成・清水等）

| 前提 | 内容 |
|------|------|
| **売上高 vs 完成工事高** | ゼネコンは連結決算で「売上高」を使用。設備会社は「完成工事高」が業界標準。混同しない |
| **選別受注が前提** | 採算の合う案件のみ。単価競争には乗らない |
| **経常利益・ROEが重要** | 売上規模より利益率。大林は自己資本比率、大成はROEを強調 |
| **洋上風力** | 鹿島・清水が注力。成長分野 |
| **公共vs民間** | 公共33%、民間67%。企業により得意分野が異なる |

### 2.2 電気設備5社（きんでん・関電工・九電工・トーエネック・ユアテック）

| 前提 | 内容 |
|------|------|
| **完成工事高で表現** | 「売上」ではなく「完成工事高」が業界標準 |
| **きんでんは電力系ではない** | 唯一の独立系。他4社は関西・九州・中部・東北電力系 |
| **九電工→クラフティア** | 2025年10月社名変更予定。会話で混乱しないよう |
| **設備工事の約50%が電気** | 建築・土木の両方に横断。DC・工場・再エネで需要増 |
| **2025年3月期は全社過去最高** | 業界全体が好調。これを知らないと会話が噛み合わない |

### 2.3 空調設備（高砂熱学・大気社・ダイダン・三機工業・新日本空調）

| 前提 | 内容 |
|------|------|
| **サブコン** | ゼネコンの下で設備工事。躯体はやらない |
| **大型産業系が成長** | 半導体・DC・工場・医療。ビル空調より成長 |
| **三機工業は電気も** | ビル空調82%、電気13%。産業空調・クリーンルームに強み |
| **大気社は塗装も** | 空調・塗装・環境システム。自動車メーカー向け塗装あり |

### 2.4 ハウスメーカー（大和・積水・住友林業）

| 前提 | 内容 |
|------|------|
| **戸建とマンションは別** | 大和・積水は戸建、長谷工はマンション。混同しない |
| **住宅以外の事業** | 大和は物流・資材、積水は注文住宅。売上構成を確認 |
| **ZEHが必須** | 省エネ基準。補助金と連動 |

## 4. 避けるべき・注意すべき話題

| 話題 | 注意点 |
|------|--------|
| **下請の単価を下げる** | 未来を拓くパートナーシップで取引適正化が政策。下請いじめの印象を与えない |
| **公共入札＝価格だけ** | 総合評価入札。価格以外も評価。技術・実績・地域貢献を軽視しない |
| **人手不足を批判的に** | 業界の構造的課題。「大変ですね」と共感し、「スキルアップの機会」と前向きに |
| **「売上」と「完成工事高」** | 設備会社は完成工事高を使う。売上と言い換えても通じるが、業界用語を知っていると信頼感 |
| **競合他社の悪口** | 業界は狭い。他社批判は避ける |
| **補助金に依存している** | 事実だが、言い方に注意。「補助金が追い風」程度に |

## 5. 会話で聞くべき・示すべきこと

### 5.1 示すべきこと（業界理解の証明）

- 「75兆円市場、民間非住宅が+12.9%でけん引」
- 「受注は過去10年最高。大手50社で20.4兆円」
- 「成長セグメントは物流・工場・DC・再エネ」

- 「骨太方針・GXの下で補助金が投資を後押し。建設関連で約4兆円超」（08参照）

## 7. 上場企業経営者との面談で必要な追加知識

上場企業の経営者（社長・IR担当役員等）と話す場合、**投資家・株主に説明している文脈**を理解していると信頼感が増す。以下を面談前に把握すること。

### 7.1 IR・決算の文脈

| 項目 | 内容 |
|------|------|
| **決算スケジュール** | 建設業界の多くは3月決算。決算発表は5月前後。業績説明会は決算後1〜2週間 |
| **四季報・決算短信** | 直近の業績・見通しは決算短信で確認。四季報で業界比較 |
| **有価証券報告書** | 事業セグメント別の収益、リスク要因、経営方針が記載 |

### 7.2 投資家・株主の関心（経営者が説明している指標）

| 指標 | 意味 | 建設業界での例 |
|------|------|----------------|
| **ROE** | 自己資本利益率。資本効率の指標 | 大成建設13.8%、大林組・きんでん等が強調 |
| **ROIC** | 投下資本利益率。投資効率の指標 | 関電工・九電工が中期目標で掲げる |
| **自己資本比率** | 財務健全性。負債に頼らない経営 | 大林組38.1%でゼネコン最高 |
| **配当性向** | 利益の何%を配当に回すか | きんでん・九電工等が40%目安 |
| **営業利益率** | 本業の稼ぐ力 | きんでん8.6%、設備会社は採算改善を強調 |

### 7.3 中期経営計画の把握

| 企業タイプ | 把握すべきこと |
|------------|----------------|
| **ゼネコン** | 選別受注、成長セグメントシフト、洋上風力・DC参入 |
| **電気設備5社** | 各社の中期計画名・数値目標（売上、ROE、ROIC等）。きんでん「Sustainable Growth 2026」、関電工「さらにかわる」等 |
| **空調設備** | 大型産業系へのシフト、採算改善、M&A・DX投資 |

※**各社の中期計画・戦略の詳細**は [10-equipment-company-strategy.md](./10-equipment-company-strategy.md)、[company-ir/companies/](./company-ir/companies/)（69社IR分析）参照。
