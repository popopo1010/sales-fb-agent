# 電気工事業界における年収相場（平均）

候補者の資格・経験に応じた年収相場。希望年収がこのレンジからずれている場合、CAは「どうすればその年収を許容させることができるのか」を考え、適切なコミュニケーションを取る必要がある。

## 資格・経験別の年収相場（万円）

| 資格・経験 | 年収レンジ | 備考 |
| --- | --- | --- |
| **資格なし** | 360〜400 | 未経験・無資格の実態 |
| **第2種電気工事士・経験なし** | 360〜420 | 資格あり・現場経験なし |
| **第2種電気工事士・現場経験あり** | 400〜500 | 2種＋実務経験 |
| **第1種電気工事士** | 450〜600 | 1種保有 |
| **施工管理の経験あり** | 600以上 | 施工管理ルート |

## 相場とのずれがある場合の対応

- 候補者の希望年収が上記相場レンジに入ってこない場合、その年収を許容させるためのコミュニケーションが必要となる
- OPSマニュアル「条件が高すぎる時（否定せず現実へ）」のフレーズを参照
- 「希望は希望、現実の勝ち筋をあわせて設計する」「選択肢を3つに分けて一緒に設計（挑戦/現実/安全）」等のアプローチを検討
- 裏のニーズ（なぜその年収を求めるのか）を深掘りし、代替で満たせる要素を提案する
