## マッキンゼー・フレームワーク構成

```

│
├─ 1. Industry Attractiveness … 業界の魅力度（5 Forces + PEST）
├─ 2. Competitive Position … 競争ポジション（3C）
├─ 3. Value Chain … バリューチェーン・利益構造

├─ 6. Business Info … 事業情報・各社サマリー

```
