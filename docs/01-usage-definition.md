| 最終更新 | 2025-02-13 |

## 2. システムの目的

人材紹介における**初回面談の書き起こし**を入力とし、**PSS（営業スキーム）**と**オペレーションマニュアル**に基づいて**フィードバック**を自動生成し、**Slack**へ送信する仕組みを提供する。

## 3. 用語定義

| 用語 | 定義 |
|------|------|
| 初回面談 | 候補者・求職者との初めての対面/オンライン面談 |
| 書き起こし | 面談の音声を文字起こししたテキスト |
| PSS | 営業スキーム（プロセス・ステップ・手法の体系） |
| FB | フィードバック（改善点・アドバイス・評価） |
| オペレーションマニュアル | 業務手順・ルール・フローを定義した文書 |

## 4. 参照ドキュメント

### 4.1 格納場所

| 種類 | 格納パス | 説明 |
|------|----------|------|
| **PSSマニュアル** | `reference/pss/` | PSS営業スキームの定義 |
| **オペレーションマニュアル** | `reference/operations/` | 業務手順・ルール・フロー |
| **プロンプトテンプレート** | `config/prompts/` | FB生成用プロンプト |

### 4.2 PSSマニュアルの格納ルール

**必須ファイル例:**
- `reference/pss/pss-sales-scheme.md` … PSSの全体像・フロー・ステップ
- `reference/pss/evaluation-criteria.md` … 評価基準・チェック項目（任意）
- `reference/operations/operation-manual.md` … オペレーションルール

**形式:**
- Markdown を推奨（AIが読みやすい）
- 画像や図表は `reference/pss/assets/` に格納

### 4.3 参照ドキュメントの利用

エージェントは FB 生成時に以下を参照する:
1. `reference/pss/` 内の全ファイル
2. `reference/operations/` 内の全ファイル
3. Cursor rules による補足指示

## 5. 利用者

| 役割 | 説明 |
|------|------|
| 営業担当 | 書き起こしを投入し、FBを受け取る |
| マネージャー | SlackでFBを確認・レビュー |
| 管理者 | システム設定・運用 |

## 6. 利用制限

- 書き起こしは機密情報を含むため、社内・権限内でのみ利用
- Slack送信先は指定チャンネル/ユーザーに限定
- PSS・オペレーションマニュアルは社外非公開
