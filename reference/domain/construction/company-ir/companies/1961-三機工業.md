# 三機工業（1961・東証プライム）企業詳細

建築設備（ビル空調・電気）大手。産業空調・クリーンルームに強み。2025年創立100周年。IR・株主向け説明資料に基づく分析。

---

## 1. 企業概要

| 項目 | 内容 |
|------|------|
| 正式名称 | 三機工業株式会社 |
| 設立 | 1961年（2025年創立100周年） |
| 本社 | 東京都港区 |
| 市場 | 東証プライム |
| 事業 | 建築設備（ビル空調衛生82%、電気13%）、環境システム、プラント設備 |
| 売上高（2025年3月期） | 2,531億円 |
| 強み | 産業空調・クリーンルーム |

**業界での位置づけ**: 空調・電気の総合設備。産業空調で差別化。

---

## 2. 経営方針・ビジョン

### 2.1 経営ビジョン「MIRAI 2030」（2025年4月策定）

- **経営理念**: 「エンジニアリングをつうじて快適環境を創造し、広く社会の発展に貢献する」
- **次の100年の使命**: 「人に快適を。地球に最適を。」
- **サステナブルな世界の実現**を目指す

### 2.2 2030年のありたい姿（定量目標）

| 指標 | 目標 |
|------|------|
| 売上高 | 3,500億円 |
| ROE | 16.0%以上 |
| 従業員数 | 3,000人 |

### 2.3 中期経営計画2027 重点テーマ

**「深化と共創」**

- 最初の3年間を「飛躍のための土台作り期間」と位置づけ
- **既存事業の深化**と**多様なパートナーとの共創**を推進
- 「選ばれ続ける三機へ」

---

## 3. 戦略の前提

### 3.1 重点戦略

| 戦略 | 内容 |
|------|------|
| コア事業の戦略的深化 | フロントローディング、LCE事業推進 |
| 脱炭素事業の推進 | ZEB、省エネ |
| デジタル技術による事業進化 | BIM、ICT活用 |
| 戦略的アライアンス | 協力会社〜スタートアップと共創 |
| 人的資本経営の推進 | 人財強化 |

### 3.2 構造課題への対応

- 技術を磨き、施工の効率化・省人化・省力化で既存事業を深化
- 協力会社〜スタートアップと共創
