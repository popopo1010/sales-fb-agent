# 新規入社者向けオンボーディング資料（Prex）

電気関係・太陽光関係・施工管理（建設）のカバー領域における実務ノウハウ

業界リファレンス（00〜18）とあわせて活用

※資格要件・必要年数は法改正で変動する場合あり

---

## 1. 事業前提・カバー領域

| 項目 | 内容 |
|------|------|
| **カバー領域** | 電気関係、太陽光関係、施工管理（建設）。施工管理はフェーズ1 |
| **取引法人** | 約1,500社。そのうちコア求人（初回で当てる求人）20社 |
| **コア求人エリア** | 東京、愛知、福岡、大阪に絞る |

---

## 2. 電気工事士「実務経験」の基本条件

| 項目 | 要件・ポイント |
|------|---------------|
| **従事できる立場** | 第二種免状もしくは認定電気工事従事者の範囲で合法的に作業（無資格作業はカウント不可） |
| **対象工事** | 電気工作物の**設置または変更工事**（配線・盤・機器取付け等） |
| **除外工事** | 軽微工事、ネオン・非常用発電装置、50kV以上架空線路、保安通信設備 |
| **経験期間** | 1日＝8h換算で**合算3年以上**。2022年改正で「5年→3年」へ短縮 |
| **証明方法** | 勤務先代表者が発行する「実務経験証明書」と工事実績一覧を提出 |

> **ポイント**: 経験に入るのは「資格を得てから」の期間。合格前の無資格作業はカウントされない。

---

### 2.1 実務経験を積める代表的パターン

| 入口資格 | 積める現場例 | 必要年数 |
|----------|--------------|----------|
| 第二種電気工事士 | 戸建・小規模店舗（50kW未満） | 合格即免状 |
| 第二種+3年 **または講習** | ビル自家用（500kW未満）低圧部 | 3年 |
| 第二種+3年 **または電験等** | 500kW以上・高圧受電設備 | 3年 |
| 電験三種免状所持者 | 工場・ビルの保安監督業務 | 5年 |

---

### 2.2 実務経験にならない主な作業

- 電球交換・コンセントカバー交換など**軽微工事**
- ネオン工事／非常用発電装置など**特殊電気工事**
- **50kV以上の架空電線路工事**
- 保安通信設備工事
- 無資格・無登録で行った違法工事一切

## 4. 実務経験証明の作成・申請の流れ

1. **証明者の確認**: 登録電気工事業者等の下で従事したことを会社印で明記
2. **工事内容を具体的に**: 「〇〇ビル受変電設備更新（契約電力1,200kW）幹線張替え」など
3. **期間集計**: 週4日勤務等は日数合算表を添付し「合計○年○月○日」と記載
4. **提出先**: 都道府県窓口（第一種・第二種）または産業保安監督部（認定・特種）

## 7. 面談フロー・トーク・提案方法

### 7.1 面談の前提

| 項目 | 内容 |
|------|------|
| **初回面談** | **企業を提案しないと失注する**（未契約求人でもあてる） |
| **複数社提案** | 1社だけだと3ヶ月でやめてしまう。2〜3社を比較検討 |
| **時間** | 一人10〜20分くらい。施工管理は忙しい |
| **普段の業務時間** | いつだったらつながるかを聞いておく |

---

### 7.2 トーク内容（施工管理）

- どんな現場で、どんな規模で、どんな人数で施工管理をやっていたか
- その対応件数をどの程度でやっていたのか
- これまでの就業期間どれくらいあるか
- 同じ業界だったらトータルどれくらいいたか
- どこのプロジェクトにいたか

---

### 7.3 提案方法・初回の流れ

**提案方法**
- コア求人を各都道府県であてる
- 必ず電話面談実施。求職者のキャリアにあっていることを口頭で説明
- 年収相場を伝え、超えている場合は「なぜ辞めたいのか？」を確認

**初回の流れ**: 電話面談①(15分) → LINE/メール → 電話面談② → 書類回収 → 雛形で空欄うめ

## 9. 事業フェーズ・立ち上げのポイント

| フェーズ | 内容 |
|----------|------|
| **Phase1** | 死ぬほど活動量担保。22:00まで架電し続ける |
| **Phase2** | 法人を絞る。求人を聞いて当てるだけに |
| **Phase3** | CAひとりあたり新規60〜70件、初回面談35件（50%） |

**立ち上げで重要なポイント ベスト3**
1. 圧倒的行動量担保（架電数100/日）
2. メイン顧客の確保：1社理解を深める
3. 法人区分け：経験/未資格、給与、未経験
