## 2. 電気工事士の実務知識（資格・経験）

### 2.1 実務経験として認められる基本条件

| 項目 | 要件・ポイント | 典拠 |
|------|---------------|------|
| **従事できる立場** | 第二種電気工事士免状もしくは認定電気工事従事者の範囲で合法的に作業していること（無資格作業はカウント不可） | 東京都電気工事工業組合 |
| **対象工事** | 電気工作物の**設置または変更工事**（配線・盤・機器取付け等）。設計・検査も含むが、製造や土木作業のみは除外 | 東京都電気工事工業組合 |
| **除外工事** | 軽微工事（電球交換など）、ネオン・非常用発電装置といった特殊電気工事、50kV以上の架空線路、保安通信設備、法令違反工事 | 東京都電気工事工業組合 |
| **経験期間の数え方** | 1日＝8h換算で**合算3年以上**（第一種電気工事士免状の場合）。パート勤務でも可。2022年改正で「5年→3年」へ短縮 | 工事士.com等 |
| **証明方法** | 勤務先（電気工事業登録業者など）の代表者が発行する「実務経験証明書」と工事実績一覧を免状申請時に提出 | - |

> **ポイント**: 経験に入るのは「資格を得てから」「合格通知を受けてから」の期間。合格前の無資格作業はカウントされない。

### 2.2 実務経験を積める代表的パターン

| 入口資格 | 積める現場例 | 必要年数 | 免状・認定申請先 |
|----------|--------------|----------|------------------|
| 第二種電気工事士 | 戸建・小規模店舗（50kW未満）の配線工事 | ―（合格即免状） | 都道府県知事 |
| 第二種+3年経験 **または講習修了** | ビル自家用（500kW未満）低圧部 | 3年 | 認定電気工事従事者を経産局へ申請 |
| 第二種+3年経験 **または電験等** | 500kW以上・高圧受電設備 | 3年 | 第一種電気工事士免状を都道府県知事へ申請 |
| 高圧電気工事技術者試験合格者 | 同上 | 3年 | 同上 |
| 電験三種免状所持者 | 工場・ビルの保安監督業務 | 5年 | 第一種電気工事士免状 |

### 2.3 実務経験にならない主な作業

- 電球交換・コンセントカバー交換など**軽微工事**
- ネオン工事／非常用発電装置（500kW未満）など**特殊電気工事**
- **50kV以上の架空電線路工事**
- 保安通信設備工事
- 無資格・無登録で行った違法工事一切

### 2.4 キャリアに応じて狙える主な資格マップ

| 分野 | 資格名 | 概要・メリット | 受験・申請要件（抜粋） |
|------|--------|----------------|------------------------|
| **施工実務** | 第二種電気工事士 | 戸建・小規模店舗の電気工事 | 受験資格なし、試験合格で免状 |
| | 第一種電気工事士 | 600V以下全般+高圧受電設備（500kW未満低圧含む） | 試験合格+実務3年以上 |
| | 認定電気工事従事者 | 500kW未満の低圧自家用工事を単独で可 | 第二種+3年経験 **または** 講習 |
| | 特種電気工事資格者 | ネオン・非常用発電装置など特殊工事 | 第二種+3年経験、または講習 |
| **施工管理** | 2級電気工事施工管理技士 | 現場代理人・主任技術者（小規模） | 19歳以上+所定実務。学歴で短縮可 |
| | 1級電気工事施工管理技士 | 監理技術者・大規模工事の統括 | 一次合格後3〜5年の実務など |
| **保安監督** | 電気主任技術者（三種） | 50kV未満受電設備の保安責任者 | 実務不要で受験可（合格後選任には会社側要件） |
| **消防・弱電** | 消防設備士 甲4・乙7 | 自動火災報知設備などの工事/点検 | 第二種所持者は筆記一部免除 |
| **通信系** | 工事担任者（AI/DD総合など） | 電話・ネット配線の端末接続工事 | 受験資格なし |
| **安全衛生** | 低圧・高圧特別教育、高所作業車など | 作業安全講習。手当対象にしやすい | 講習受講で取得 |

> **Tip**: 近年はEV充電器設置や再エネ事業で高圧・特高案件が増加。早めに第一種と1級施工管理技士を組み合わせると仕事の幅が広がる。

### 2.5 実務経験証明の作成・申請の流れ

1. **証明者の確認**: 登録電気工事業者や電気主任技術者の下で従事した期間であることを会社印・証明者印で明記
2. **工事内容を具体的に**: 「〇〇ビル受変電設備更新（契約電力1,200kW）幹線張替え・動力盤取替」など、設置or変更であることが分かるよう記載
3. **期間集計**: 週4日勤務などの場合は日数合算表を添付し「合計○年○月○日」と記載
4. **提出先**: 都道府県窓口（第一種・第二種免状）または産業保安監督部（認定電気工事従事者・特種）

## 5. 面談・提案・障壁突破

### 5.1 面談の前提

| 項目 | 内容 |
|------|------|
| **初回面談** | **企業を提案しないと失注する**（未契約求人でもあてる/duda） |
| **複数社提案** | 1社だけだと3ヶ月でやめてしまう。2〜3社を比較検討した方がよい |
| **時間** | 一人10〜20分くらい。施工管理は忙しい |
| **普段の業務時間** | いつだったらつながるかを聞いておくこと |

### 5.2 トーク内容（施工管理）

- どんな現場で、どんな規模で、どんな人数で施工管理をやっていたか
- その対応件数をどの程度でやっていたのか
- これまでの就業期間どれくらいあるか
- 同じ業界だったらトータルどれくらいいたか
- どこのプロジェクトにいたか

### 5.3 提案方法

| 項目 | 内容 |
|------|------|
| **コア求人** | 各都道府県でコア求人をあてることが重要 |
| **求人紹介** | 必ず電話面談実施。業界特徴・会社特徴・背景を踏まえ、求職者のキャリアにあっていることを口頭で説明 |
| **年収相場** | 相場を伝え、相場を超えている場合は「なぜ辞めたいのか？」を確認。レンジまで落とせるなら対応 |
| **単身赴任** | 年収高くないといけない。実家になるといける |

### 5.4 初回の流れ

1. 電話面談①：15分
2. 「合う求人紹介します」でLINE or メール
3. 「求人の特長説明させてください」で電話面談②を取る
4. 書類回収（上からやる）
5. 雛形を送って空欄うめてもらう / LINEで全項目送って回答 / 電話で全項目ヒアリングしてこちらで作る
6. ※回収できない方は別に原因ありと考える

### 5.5 障壁別ヒアリングの具体的な言い回し

求職者が前に進めない「障壁」に対して、何を聞き、どう言えばよいかを具体的に示す。

※FBシステム（Slack /fb）の残論点生成時は OPS [barrier-interview-questions.md](../../operations/barrier-interview-questions.md) を参照。本セクションは建設特化の言い回し（ゼネコン・第二種・施工管理等）を含む。

#### 転職時期・動き出しの障壁

| 障壁 | 聞く目的 | 具体的な言い回し（例） |
|------|----------|------------------------|
| **転職希望月が固い** | 本当に動けないのか、条件次第で前倒しできるか | 「〇月まで待つ理由は、今の現場の納品とかありますか？」「もし条件の合う求人が早く出たら、検討の余地はありますか？」 |
| **現場終わるまで動けない** | いつ頃なら動けるか、中途入場の余地はあるか | 「今の現場はいつ頃までですか？」「現場の引き継ぎが終わったら、すぐ動けそうですか？」「途中で別の人に引き継いでもらう形は、現実的には難しいですか？」 |
| **CV時点で動かせるか不明** | 本気度・優先度を測る | 「いま転職活動、どのくらいの優先度で考えていますか？」「良い求人があったら、すぐ動ける状態ですか？それとも〇月以降になりますか？」 |

#### 転居・勤務地の障壁

| 障壁 | 聞く目的 | 具体的な言い回し（例） |
|------|----------|------------------------|
| **転居したくない** | 単身赴任・週末帰宅・実家近くなら可か | 「転居は難しいとのことですが、単身赴任なら検討できますか？」「実家が近いエリアなら、単身赴任しやすいですか？」「週末は帰れる前提なら、どのくらいの距離までなら考えられますか？」 |
| **家族がいる（再エネ・転勤案件）** | 家族の理解、条件次第で動けるか | 「ご家族の転居や単身赴任について、どの程度まで話し合えていますか？」「年収やキャリアのメリットが大きければ、家族で検討する余地はありますか？」 |
| **勤務地の希望が狭い** | 妥協できる範囲、理由を把握 | 「〇〇エリアにこだわる理由はありますか？（通勤時間、家族、住居など）」「電車で〇分以内など、妥協できる範囲はありますか？」 |

#### 年収・条件の障壁

| 障壁 | 聞く目的 | 具体的な言い回し（例） |
|------|----------|------------------------|
| **年収希望が相場より高い** | 辞めたい理由、譲れる幅 | 「いまの年収より上を希望されている理由を教えてもらえますか？」「転職の目的が叶うなら、相場レンジまで下げることは検討できますか？」 |
| **年収は満足している** | 転職の本気度、本当の動機 | 「年収に不満がないとのことですが、転職を考えている理由は何でしょうか？」「キャリアや働き方で、いま一番叶えたいことはありますか？」 |
| **休み・労働条件へのこだわり** | 譲れないライン、妥協できる点 | 「年間休日や土曜出勤について、どこまでが譲れないラインですか？」「建設業だと土曜出勤がある会社が多いですが、月に何回までなら許容できますか？」 |

#### 資格・経験の障壁

| 障壁 | 聞く目的 | 具体的な言い回し（例） |
|------|----------|------------------------|
| **実務経験が不明・曖昧** | 資格取得後の経験年数、対象工事か | 「第二種を取ってから、何年くらい電気工事に携わっていますか？」「配線工事や盤の取替えなど、設置・変更工事をされていますか？電球交換だけだと経験に含まれないんです」 |
| **未経験・未資格** | 受験予定、講習の検討有無 | 「第一種や施工管理技士の受験は予定されていますか？」「認定講習を受けて資格を取る方向で検討されていますか？」 |
| **ゼネコン経験がない** | 他社経験でカバーできるか、狙う企業の幅 | 「ゼネコン以外の施工管理経験は何年ありますか？」「50〜100名規模の設備会社や専門工事業者なら、検討できますか？」 |

#### 連絡・書類回収の障壁

| 障壁 | 聞く目的 | 具体的な言い回し（例） |
|------|----------|------------------------|
| **連絡がつきにくい** | つながりやすい時間帯、手段 | 「現場だと電話に出づらいですよね。いつ頃ならお話しできますか？昼休みや終業後など」「LINEやメールの方が返しやすいですか？」 |
| **書類が回収できない** | 忙しさ、不安、手間のどれか | 「書類のご提出が進んでいないようなのですが、何かお困りですか？」「空欄を埋めるのが大変なら、電話でヒアリングして私が作成することもできます」 |
| **返信が遅い・途切れる** | 本気度、優先度、別の不安 | 「最近お忙しいようで、転職の優先度はどのくらいですか？」「何か気になる点や不安があれば、遠慮なく教えてください」 |

#### ヒアリングの基本姿勢

- **詰問しない**: 「なぜ〇〇なんですか？」より「〇〇の理由を教えてもらえますか？」
- **選択肢を出す**: 「単身赴任は難しいですか？それとも実家近くなら考えられますか？」のように二者択一で聞くと答えやすい
- **譲歩の余地を探る**: 「絶対無理」と決めつけず、「条件次第で」「〇〇なら」の可能性を聞く

## 7. 事業フェーズ・立ち上げのポイント

### 7.1 フェーズ別

| フェーズ | 内容 |
|----------|------|
| **Phase1** | 死ぬほど活動量担保して獲得しにいく。22:00まで架電し続ける |
| **Phase2** | 強い人が他社でも決定しなかった。名前があるところでやってもうまくいかなかった。900件CVを取っても各オーダー理解ができない。求人を聞いて当てるだけに。→法人を絞る |
| **Phase3** | 分母が減るが。CAひとりあたり新規60〜70件、そのうち初回面談35件（50%） |

### 7.2 事業立ち上げで重要なポイント ベスト3

1. **圧倒的行動量担保**（架電数100/日）
2. **メイン顧客の確保**: 徹底的に商談して1社理解を深める。数取れば取るほど沼
3. **法人区分け**: 経験ある/未資格、50〜60代遅れる、給与がいい、未資格/未経験

### 7.3 法人への条件緩和

- 資格持っていて経験ある人を求めている → 資格はあるけど未経験も取る
- 中堅取れていると若手も取り入れたほうがいい
- 未通電の対応
